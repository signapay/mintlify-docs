---
title: "Agents Overview"
description: "Manage sales agents and their merchant assignments"
---

Agents are sales representatives who originate and manage merchant leads on behalf of a partner organization. Each agent record tracks identity, contact details, and status within the SignaPay network.

## What's included

### Core

- **Agent records** — List, create, retrieve, and update agent profiles.
- **Status transitions** — Activate, deactivate, or close an agent via `PUT /agents/{agentId}/{action}`.
- **Sensitive data** — Decrypt protected fields (e.g. SSN, bank account numbers) on demand via the dedicated decrypt endpoint.

### Sub-resources

- **Documents** — Attach, retrieve, update, and remove documents associated with an agent.
- **Notes** — Add and manage internal notes on agent records.
