title: "Overview"
description: "Aggregated metrics and visualizations across the platform"
---

The Dashboard endpoints return pre-aggregated metrics that power the SignaPay admin, partner, agent, and merchant UIs. All responses are read-only and scoped to the authenticated user's role and client hierarchy.

## Dashboard types

### Agent dashboard

Metrics scoped to a specific agent's portfolio:

| Endpoint | Description |
| --- | --- |
| [Applications summary](/api-reference/dashboards/agent-applications-summary) | Application counts broken down by pipeline stage |
| [KPI metrics](/api-reference/dashboards/agent-kpi-metrics) | Key performance indicators for the agent |
| [Merchant status](/api-reference/dashboards/agent-merchant-status) | Active, inactive, and closed merchant breakdown |
| [Monthly active merchants](/api-reference/dashboards/agent-monthly-active-merchants) | Month-over-month active merchant count |
| [Monthly processing volume](/api-reference/dashboards/agent-monthly-processing-volume) | Rolling monthly processing volume trend |
| [Net merchants](/api-reference/dashboards/agent-net-merchants) | Net merchant adds and attrition by month |
| [Ticket counts](/api-reference/dashboards/agent-ticket-counts) | Open ticket summary for the agent |
| [Top performers (processing)](/api-reference/dashboards/agent-top-performers-processing) | Highest-volume merchants in the agent's book |

### Agent residuals

ISO residual data scoped to an agent's portfolio. These endpoints are served from the [Financial Reporting](/api-reference/financial-reporting/overview) section:

| Endpoint | Description |
| --- | --- |
| [List residuals](/api-reference/financial-reporting/list-residuals) | ISO residual historical summaries and aggregated metrics |
| [Get residual](/api-reference/financial-reporting/get-residual) | Single residual period summary |
| [Get residual metrics](/api-reference/financial-reporting/get-residual-metrics) | Aggregated residual metrics |
| [List all merchant summaries](/api-reference/financial-reporting/list-all-residual-merchant-summaries) | All-time merchant-level residual summaries |
| [List merchant summaries](/api-reference/financial-reporting/list-residual-merchant-summaries) | Merchant summaries for a specific residual period |
| [Get merchant summary](/api-reference/financial-reporting/get-residual-merchant-summary) | Single merchant residual summary |
| [List gross details](/api-reference/financial-reporting/list-residual-gross-details) | Gross residual detail records |
| [List merchant details](/api-reference/financial-reporting/list-residual-merchant-details) | Line-item residual details for a merchant |
| [Get merchant detail](/api-reference/financial-reporting/get-residual-merchant-detail) | Single merchant residual detail record |
| [List all overrides](/api-reference/financial-reporting/list-all-residual-overrides) | All-time residual overrides |
| [List overrides](/api-reference/financial-reporting/list-residual-overrides) | Overrides for a specific residual period |
| [Get override](/api-reference/financial-reporting/get-residual-override) | Single residual override record |
