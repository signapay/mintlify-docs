---
title: "Clear Identity Verification"
description: "Initiate and check Clear identity verification sessions for a lead"
---

Clear is a biometric identity verification provider. Use these endpoints to initiate a verification session for a lead principal and retrieve the verification outcome.

| Endpoint | Description |
| --- | --- |
| [Create Clear session](/api-reference/leads/create-clear-session) | Initiate a Clear identity verification session for a lead |
| [Get Clear status](/api-reference/leads/get-clear-verification-status) | Retrieve the result of a Clear verification |
