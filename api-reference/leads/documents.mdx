---
title: "Documents"
description: "Manage documents attached to a lead record"
---

Lead documents store uploaded files required during the underwriting process, such as voided checks, government IDs, and signed agreements.

| Endpoint | Description |
| --- | --- |
| [List documents](/api-reference/leads/list-lead-documents) | Retrieve all documents for a lead |
| [Create document](/api-reference/leads/create-lead-document) | Attach a document to a lead |
| [Get document](/api-reference/leads/get-lead-document) | Retrieve a single lead document |
| [Update document](/api-reference/leads/update-lead-document) | Update document metadata |
| [Delete document](/api-reference/leads/delete-lead-document) | Remove a document from a lead |
