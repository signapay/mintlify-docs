---
title: "Fast Apply"
description: "Streamlined public-facing MPA form endpoints for lead capture"
---

Fast Apply is a simplified, public-facing merchant application form designed for low-friction lead capture. It accepts partial data across three steps — general information, business details, and principals — before a full lead record is created.

| Endpoint | Description |
| --- | --- |
| [Update general](/api-reference/leads/update-fast-apply-general) | Update the general section of a Fast Apply form |
| [Update business](/api-reference/leads/update-fast-apply-business) | Update the business details section |
| [Save principals](/api-reference/leads/save-fast-apply-principals) | Save the principals section |
| [Create principal](/api-reference/leads/create-fast-apply-principal) | Add a principal via the Fast Apply form |
| [Update principal](/api-reference/leads/update-fast-apply-principal) | Update a specific principal via the Fast Apply form |
