---
title: "Lead Sources Overview"
description: "Track referral channels and marketing campaigns that originate leads"
---

Lead sources identify the referral channel, partner, or campaign that originated a lead. Each source can be associated with multiple leads, enabling attribution reporting and pipeline analysis by source.

| Endpoint | Description |
| --- | --- |
| [List lead sources](/api-reference/leads/list-lead-sources) | Retrieve all lead sources |
| [Create lead source](/api-reference/leads/create-lead-source) | Create a new referral or campaign source |
| [Get lead source](/api-reference/leads/get-lead-source) | Retrieve a single lead source by ID |
| [Update lead source](/api-reference/leads/update-lead-source) | Update an existing lead source |
| [Get source summary](/api-reference/leads/get-lead-source-summary) | Retrieve an aggregate summary for a lead source |
| [List leads for source](/api-reference/leads/list-leads-for-source) | Retrieve all leads associated with a lead source |
| [List sources for lead](/api-reference/leads/list-lead-lead-sources) | Retrieve all sources associated with a lead |
| [Add source to lead](/api-reference/leads/add-lead-source-to-lead) | Associate a lead source with a lead |
| [Remove source from lead](/api-reference/leads/remove-lead-source-from-lead) | Remove a lead source association from a lead |
