---
title: "Pricing Overview"
description: "Manage pricing items, packages, and rules for an application"
---

Application pricing captures the fee structure that will be applied when the application is boarded as a merchant. Items define individual fee line items, packages group items into reusable bundles, and rules define conditional fee logic.

## Pricing items

| Endpoint | Description |
| --- | --- |
| [List pricing items](/api-reference/leads/list-lead-pricing-items) | Retrieve all pricing items for an application |
| [Create pricing item](/api-reference/leads/create-lead-pricing-item) | Add a pricing item to an application |
| [Get pricing item](/api-reference/leads/get-lead-pricing-item) | Retrieve a single pricing item |
| [Update pricing item](/api-reference/leads/update-lead-pricing-item) | Update an application pricing item |
| [Delete pricing item](/api-reference/leads/delete-lead-pricing-item) | Remove a pricing item from an application |
