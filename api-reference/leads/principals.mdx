---
title: "Principals Overview"
description: "Manage beneficial owners and signatories on an application record"
---

Principals are the individuals who own, control, or are responsible for the business applying for merchant processing. Each principal record captures personal identity details used for KYC and compliance screening, with sensitive fields stored encrypted.

| Endpoint | Description |
| --- | --- |
| [List principals](/api-reference/leads/list-lead-principals) | Retrieve all principals associated with an application |
| [Add principal](/api-reference/leads/add-lead-principal) | Add a beneficial owner or signer to an application |
| [Get principal](/api-reference/leads/get-lead-principal) | Retrieve a single principal by ID |
| [Update principal](/api-reference/leads/update-lead-principal) | Update an application principal's information |
| [Delete principal](/api-reference/leads/delete-lead-principal) | Remove a principal from an application record |
| [Decrypt principal data](/api-reference/leads/decrypt-lead-principal) | Decrypt a sensitive field on a principal |
