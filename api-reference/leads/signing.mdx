---
title: "Signing Overview"
description: "Manage e-signature sessions for merchant application agreements"
---

Signing sessions deliver the MPA (Merchant Processing Agreement) to applications for electronic signature. Create a session to generate a signing link, and verify a token to confirm a session's validity.

| Endpoint | Description |
| --- | --- |
| [Create signing session](/api-reference/leads/create-signing-session) | Create an e-signature session for MPA agreement delivery |
| [Verify signing token](/api-reference/leads/verify-signing-token) | Verify a signing session token |
