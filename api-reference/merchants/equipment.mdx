---
title: "Equipment"
description: "Manage terminal equipment models and deployment orders for a merchant"
---

Equipment records define the hardware a merchant uses for card acceptance. Models specify the terminal type and configuration; orders track the physical fulfillment and deployment of that equipment, including line items, gateway settings, and file build types.

## Equipment models

| Endpoint | Description |
| --- | --- |
| [List equipment models](/api-reference/merchants/list-merchant-equipment-models) | Retrieve all equipment models for a merchant |
| [Create equipment model](/api-reference/merchants/create-merchant-equipment-model) | Add an equipment model to a merchant |
| [Get equipment model](/api-reference/merchants/get-merchant-equipment-model) | Retrieve a single equipment model |
| [Update equipment model](/api-reference/merchants/update-merchant-equipment-model) | Update a merchant equipment model |
| [Delete equipment model](/api-reference/merchants/delete-merchant-equipment-model) | Remove an equipment model from a merchant |
