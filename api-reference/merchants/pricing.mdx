---
title: "Pricing"
description: "Manage pricing items, packages, templates, and rules for a merchant"
---

Merchant pricing captures the fee structure applied to a merchant's processing. Items define individual fee line items, packages group items into bundles, templates provide reusable pricing configurations, and rules define conditional fee logic.

## Pricing items

| Endpoint | Description |
| --- | --- |
| [List pricing items](/api-reference/merchants/list-merchant-pricing-items) | Retrieve all pricing items for a merchant |
| [Create pricing item](/api-reference/merchants/create-merchant-pricing-item) | Add a pricing item to a merchant |
| [Get pricing item](/api-reference/merchants/get-merchant-pricing-item) | Retrieve a single pricing item |
| [Update pricing item](/api-reference/merchants/update-merchant-pricing-item) | Update a merchant pricing item |
| [Delete pricing item](/api-reference/merchants/delete-merchant-pricing-item) | Remove a pricing item from a merchant |
