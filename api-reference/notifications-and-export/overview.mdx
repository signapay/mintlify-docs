---
title: "Notifications and Export Overview"
description: "Send notifications and export data from the platform"
---

The Notifications and Export endpoints manage in-app notifications for the authenticated user's account and provide bulk data export for platform records.

## What's included

### Notifications

- **List notifications** — Retrieve the notification feed for the current account.
- **Get a notification** — Retrieve a single notification by ID.

### Notification settings

- **List settings** — Retrieve the current notification preferences for the account.
- **Update settings** — Enable or disable notification types or channels.

### Export

- **Export data** — Request a bulk export of platform records.
- **Export residuals** — Request a bulk export of residual report data.
