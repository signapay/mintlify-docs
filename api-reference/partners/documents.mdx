---
title: "Documents"
description: "Manage documents attached to a partner record"
---

Partner documents store files associated with a partner organization, including agreements, compliance filings, and supporting documentation.

| Endpoint | Description |
| --- | --- |
| [List documents](/api-reference/partners/list-partner-documents) | Retrieve all documents for a partner |
| [Create document](/api-reference/partners/create-partner-document) | Attach a document to a partner |
| [Get document](/api-reference/partners/get-partner-document) | Retrieve a single partner document |
| [Update document](/api-reference/partners/update-partner-document) | Update document metadata |
| [Delete document](/api-reference/partners/delete-partner-document) | Remove a document from a partner |
