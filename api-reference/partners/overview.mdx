---
title: "Partners Overview"
description: "Manage partner organizations in the SignaPay network"
---

Partners are ISO or referral organizations that bring merchants and agents onto the platform. They sit in the client hierarchy above merchants and below the platform operator.

## What's included

| Endpoint | Description |
| --- | --- |
| [List partners](/api-reference/partners/list-partners) | Retrieve a paginated list of partners |
| [Get partner](/api-reference/partners/get-partner) | Retrieve a single partner by ID |
| [Update partner](/api-reference/partners/update-partner) | Update an existing partner record |
| [Update status](/api-reference/partners/update-partner-status) | Transition a partner to a new status (activate, deactivate, close) |
| [Decrypt sensitive data](/api-reference/partners/decrypt-partner) | Decrypt a protected field on demand |
