---
title: "Principals Overview"
description: "Manage beneficial owners and key contacts on a partner record"
---

Principals are the individuals who own or control the partner organization. Each record stores personal and contact information required for compliance, with sensitive fields encrypted at rest.

| Endpoint | Description |
| --- | --- |
| [List principals](/api-reference/partners/list-partner-principals) | Retrieve all principals associated with a partner |
| [Add principal](/api-reference/partners/add-partner-principal) | Add a beneficial owner or key contact |
| [Get principal](/api-reference/partners/get-partner-principal) | Retrieve a single principal by ID |
| [Update principal](/api-reference/partners/update-partner-principal) | Update a partner principal's information |
| [Decrypt principal data](/api-reference/partners/decrypt-partner-principal) | Decrypt a sensitive field on a principal |
