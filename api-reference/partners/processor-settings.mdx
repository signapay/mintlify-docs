---
title: "Processor Settings"
description: "Manage processor-specific configuration for a partner"
---

Processor settings store the partner-level configuration required to communicate with each payment processor. Each setting entry is keyed by processor and contains the credentials or parameters that downstream merchants will inherit.

| Endpoint | Description |
| --- | --- |
| [List processor settings](/api-reference/partners/list-partner-processor-settings) | Retrieve all processor-specific settings for a partner |
| [Update processor setting](/api-reference/partners/update-partner-processor-setting) | Update a specific processor setting entry |
