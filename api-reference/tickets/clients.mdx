---
title: "Client Associations"
description: "Link and manage client records associated with a ticket"
---

Tickets can be linked to one or more client records in the hierarchy, enabling support teams to scope work items to specific merchants, partners, or agents.

| Endpoint | Description |
| --- | --- |
| [List clients](/api-reference/tickets/list-ticket-clients) | Retrieve all clients linked to a ticket |
| [Add client](/api-reference/tickets/add-ticket-client) | Associate a client with a ticket |
| [Remove client](/api-reference/tickets/remove-ticket-client) | Remove a client association from a ticket |
