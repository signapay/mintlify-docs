---
title: "External Users"
description: "Grant and manage external user access to a support ticket"
---

External users are non-platform participants given limited access to a specific ticket, such as a merchant's representative or a third-party vendor involved in resolving the issue.

| Endpoint | Description |
| --- | --- |
| [List external users](/api-reference/tickets/list-ticket-external-users) | Retrieve all external users on a ticket |
| [Add external user](/api-reference/tickets/add-ticket-external-user) | Add a non-platform user to a ticket |
| [Remove external user](/api-reference/tickets/remove-ticket-external-user) | Remove an external user from a ticket |
