---
title: "SAM Action Logs Overview"
description: "Retrieve audit log entries for platform entities"
---

SAM (Status / Action / Message) action logs provide a timestamped audit trail of status changes and significant actions taken on any platform entity. Query these logs to review the history of a merchant, partner, lead, or other resource.

| Endpoint | Description |
| --- | --- |
| [List SAM action logs](/api-reference/tickets/list-sam-action-logs) | Retrieve SAM audit log entries for any platform entity |
