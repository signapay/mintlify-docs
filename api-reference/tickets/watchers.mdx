---
title: "Watchers Overview"
description: "Manage notification subscribers on a support ticket"
---

Watchers are users subscribed to receive notifications when a ticket is updated. Adding a watcher ensures they are alerted to activity, comments, and status changes on the ticket.

| Endpoint | Description |
| --- | --- |
| [List watchers](/api-reference/tickets/list-ticket-watchers) | Retrieve all watchers on a ticket |
| [Add watcher](/api-reference/tickets/add-ticket-watcher) | Subscribe a user to ticket notifications |
| [Remove watcher](/api-reference/tickets/remove-ticket-watcher) | Unsubscribe a user from ticket notifications |
