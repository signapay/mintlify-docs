---
title: "Authentication"
description: "How to authenticate API requests using an API key"
---

Every protected API endpoint requires an `Authorization` header with a valid API key.

```http
Authorization: Api-Key <api-key>
```

API keys are managed via the [Client API Key](/api-reference/clients/get-client-api-key) endpoints.

---

## Obtain an API key

API keys are scoped to a client account. Use the [Get Client API Key](/api-reference/clients/get-client-api-key) endpoint to retrieve the key for a given client.

---

## Make an authenticated request

Include the key in the `Authorization` header of every request.

```http
GET /v1/crm/clients
Authorization: Api-Key <api-key>
```
